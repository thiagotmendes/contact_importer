```
csv-test-file
```

Install Laravel

``` composer install ```

Install UI

``` npm install ```

create a .env file to configure database

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=database
DB_USERNAME=root
DB_PASSWORD=
```

If have problems with composer install use: 

```
COMPOSER_MEMORY_LIMIT=-1 composer install
```
